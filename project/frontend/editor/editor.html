📖 Slim Синтаксис - Ръководство

Основни команди

#title - Заглавие на слайд
#type - Тип на слайд
#data - Данни на слайд
#slide - Нов слайд

Типове слайдове

  • title - Заглавен слайд
  • text-only - Само текст
  • image-text - Изображение + текст
  • image-left - Изображение отляво
  • image-right - Изображение отдясно
  • two-column - Две колони
  • code - Код блок
  • list - Списък

Примери

Заглавен слайд

#title Въведение в Web Technologies
#type title
#data subtitle=Лекция 1;author=Име Фамилия

Текстов слайд

#slide
#title HTML Основи
#type text-only
#data content=HTML е маркиращ език за създаване на уеб страници.

Слайд с изображение

#slide
#title CSS Grid Layout
#type image-text
#data image=grid.png;text=Grid система за двумерни лейаути
